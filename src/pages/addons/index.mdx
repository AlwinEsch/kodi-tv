# Add-Ons

## Kodi can do so much more than what comes in the box

We have a huge catalogue of extra Add-On components for you to take advantage of. All Add-Ons can be installed via the application itself and they will automatically update as new versions are released. It is kind of like an "App Store" for Kodi, but everything is **free!**

[Music Add-Ons](/addons/category/music-addons) and [Video Add-Ons](/addons/category/video-addons) let you stream internet content. [Skins](/addons/category/skins) allow you to change the look and experience of the interface. [Program Add-Ons](/addons/category/program-addons) and [Service Add-Ons](/addons/category/services) provide helpful tools while [web interfaces](/addons/category/web-interfaces) are for controlling Kodi remotely. And that is just the tip of the iceberg. Browse all the categories below to see what is available and install them via Kodi.

  query {
    Games: allCategory(sort: {fields: name, order: ASC}, filter: {grouping: {eq: "Games"}}) {
      edges {
        node {
          name
          slug
          icon
        }
      }
    }
    InformationProviders: allCategory(sort: {fields: name, order: ASC}, filter: {grouping: {eq: "Information providers"}}) {
      edges {
        node {
          name
          slug
          icon
        }
      }
    }
    LookAndFeel: allCategory(sort: {fields: name, order: ASC}, filter: {grouping: {eq: "Look and feel"}}) {
      edges {
        node {
          name
          slug
          icon
        }
      }
    }
    Others: allCategory(sort: {fields: name, order: ASC}, filter: {grouping: {eq: "Other"}}) {
      edges {
        node {
          name
          slug
          icon
        }
      }
    }
    FeaturedAddons: allAddon(filter: {featured: {eq: "true"}}) {
      edges {
        node {
          name
          slug
          icon
        }
      }
    }
    NewAddons: allAddon(limit: 5, filter: {agetype: {eq: "new"}}, sort: {fields: firstseen, order: DESC}) {
      edges {
        node {
          name
          slug
          icon
        }
      }
    }
    UpdatedAddons: allAddon(limit: 5, filter: {agetype: {eq: "existing"}}, sort: {fields: lastupdate, order: DESC}) {
      edges {
        node {
          name
          slug
          icon
        }
      }
    }
  }
